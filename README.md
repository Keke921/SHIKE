## Long-Tailed Visual Recognition via Self-Heterogeneous Integration with Knowledge Excavation

This is the source code for SHIKE.

### An early demo of SHIKE for CIFAR100-LT

#### Requirements
```
python  3.7.7
torch   1.11.0
```

#### Usage
For CIFAR100-LT with an imbalanced factor of 100, please directly run the training file.

### Implementation for other datasets is under reoganizing...
